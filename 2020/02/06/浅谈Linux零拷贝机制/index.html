什么是零拷贝

零拷贝指的是数据从一个存储区域到另一个存储区域的过程中不需要cpu参与执行拷贝的动作，通常用于网络传输文件时减少cpu的负载，解放cpu去提升系统整体性能。

谈到零拷贝我们还需要了解到DMA是什么：
DMA（Direct Memory Access）是指在现代计算机系统上，外接设备可以不用CPU干预，直接把数据传输到内存的技术。正常情况下，CPU全权负责内存的读写操作，而DMA技术可以把CPU解放出来，这将使计算机性能得到显著改善。
